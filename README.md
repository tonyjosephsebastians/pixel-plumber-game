# Pixel Plumber Adventure

A fully original browser-based 2D side-scrolling platformer built with **HTML5 Canvas + vanilla JavaScript**.

## Run
1. Open `index.html` in a desktop browser.
2. Press **Enter** to start.

## Controls
- Move: **A/D** or **ArrowLeft/ArrowRight**
- Jump: **W**, **ArrowUp**, or **Space**
- Pause: **P**
- Restart: **R**

## Notes
- No external assets are used. Graphics are canvas shapes and sounds are generated with Web Audio API beeps.
- Level data is in `game.js` (`levelData` array) for easy editing.
